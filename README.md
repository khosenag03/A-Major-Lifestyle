    <td>Project Purpose</td>
    <td>What this project is about and what it aims to accomplish?

Aims to analyze the living expense data within 5 regions to identify trends in the historic data that correlate to Mid-Career salary and its relationship to the region. With machine learning, this project seeks to predict the region in which a user lives by examining the historical data of residents within a select region and the average living expenses in the selected region. 

</td>
  </tr>
  <tr>
    <td>The Need for this Project</td>
    <td>Explain the problem that the project addresses specifically.

This project provides a tool that highlights the average cost of living by region for families who are seeking to live a comfortable lifestyle based upon income. Based upon a users mid-career salary and monthly expenses, we predict which region they are most-likely to live in.</td>
  </tr>
  <tr>
    <td></td>
    <td>What evidence is there to support the need for this project?

Many students are graduating with college degrees with low income return on their college tuition investment. Providing insight for prospective students on the average cost of living per region will assist in a students decision to declare a major that will result in a lifestyle they could financially afford.</td>
  </tr>
</table>
